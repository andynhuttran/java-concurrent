# Java Concurrency

  Thread is created by 
  
      Thread t = new Thread();
  
  Start a thread
  
      t.start();
      method run() is called when a thread is started, method run comes from Thead class or 
      Runnable interface which is pass to Thead at initialization, 
      
  Wait a thread until it done
  
      t.join();
